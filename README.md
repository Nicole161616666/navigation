# 导航链接管理器

一个纯前端的链接管理应用，用于快速访问常用网站和资源。

## 功能特性

### 链接管理
- **添加链接**：填写名称、URL、分类、标签，支持表单校验
- **编辑链接**：修改已有链接的所有信息
- **删除链接**：确认后删除，支持批量删除
- **访问链接**：点击卡片或URL即可跳转，自动记录访问次数

### 分类与标签
- **自动分类**：输入新分类名称自动创建
- **分类筛选**：顶部工具栏显示所有分类按钮，支持快速筛选
- **标签系统**：多个标签用英文逗号分隔，显示带#前缀的标签
- **标签搜索**：搜索框会同时匹配标签内容

### 搜索与筛选
- **关键词搜索**：实时搜索链接名称、URL、标签
- **分类筛选**：点击分类按钮切换显示
- **组合筛选**：搜索和分类筛选可叠加使用

### 排序与置顶
- **收藏置顶**：已收藏的链接排在前面
- **点击排序**：同组内按点击次数降序排列

### 访问统计
- **点击计数**：每次访问自动+1，实时显示
- **最近访问**：顶部横向展示最近10条访问记录
- **访问趋势**：每个卡片底部显示近7天访问频次柱状图

### 批量操作
- **批量模式**：点击工具栏"批量"按钮开启
- **批量选择**：勾选多个链接，实时显示已选数量
- **批量移动**：将选中链接移动到指定分类
- **批量删除**：删除所有选中链接

### 数据持久化
- **自动存储**：所有操作自动保存至localStorage
- **导出备份**：导出为teamNav_backup_YYYY-MM-DD.json文件
- **导入备份**：支持文件导入和文本导入，覆盖前需确认

### 界面体验
- **暗黑模式**：点击右上角🌙/☀️切换主题，偏好自动保存
- **键盘快捷键**：Ctrl+K（或Cmd+K）快速聚焦搜索框
- **响应式布局**：适配桌面、平板、手机屏幕
- **空状态提示**：无数据或无搜索结果时显示友好提示

## 技术栈
- 前端：纯 HTML5 + CSS3 + JavaScript（ES6+）
- 数据存储：浏览器 localStorage
- 依赖：无第三方库（纯原生实现）
- 部署方式：单 HTML 文件，可放置于任何静态 Web 服务器或直接本地打开

## 使用说明

### 安装与运行
1. 直接打开 `index.html` 文件即可使用
2. 或部署到任何静态 Web 服务器

### 操作指南

#### 添加链接
1. 点击左侧「+ 添加链接」按钮
2. 填写链接名称、URL、分类和标签
3. 点击「保存」

#### 编辑链接
1. 将鼠标悬停在链接卡片上
2. 点击编辑按钮（✏️）
3. 修改信息后点击「保存」

#### 删除链接
1. 将鼠标悬停在链接卡片上
2. 点击删除按钮（🗑️）
3. 确认删除

#### 收藏链接
1. 将鼠标悬停在链接卡片上
2. 点击收藏按钮（⭐）切换收藏状态

#### 批量操作
1. 点击工具栏「批量」按钮进入批量模式
2. 使用复选框选择多个链接
3. 使用批量操作栏进行移动或删除

#### 筛选链接
1. 使用搜索框按名称、URL 或标签搜索
2. 点击分类按钮筛选分类
3. 点击标签筛选链接

#### 导入/导出数据
1. 点击左侧「⚙️ 设置」按钮
2. 选择「导出数据」下载备份文件
3. 选择「导入数据」上传备份文件或粘贴 JSON 数据

#### 切换主题
- 点击右上角 🌙/☀️ 按钮切换明暗主题

#### 快捷键
- Ctrl+K / Cmd+K：聚焦搜索框
